# ✉️ Courriers — Index (🔑 Token)

Ce dossier rassemble **tous les modèles de courriers** du projet Accident Main, en version **Token** (anonymisée, éditable).  
Les versions réelles correspondantes sont générées automatiquement dans `⚖️ Actes/👤 Reel/✉️ Courriers/` par `generate_real_versions.py` (cf. `🧠 Memory/RULES.md` règle #22).

Pour visualiser l'ordre logique d'expédition et les interdépendances, consultez le [Graphe des Dépendances](../../../%F0%9F%A7%A0%20Memory/DEPENDANCES.md).

```
✉️ Courriers/

```

## Rappel des règles Token/Reel

- Toute modification de contenu se fait **exclusivement** dans `⚖️ Actes/🔑 Token/`.
- Les versions réelles (données désanonymisées) sont régénérées depuis Token :

  ```bash
  python3 .dev/app/generate_real_versions.py
  python3 .dev/app/check_consistency.py
  ```

- **Aucune édition manuelle** n'est autorisée dans `⚖️ Actes/👤 Reel/` (cf. `🧠 Memory/RULES.md` règle #22).
- Les README d'index vivent côté Token (`🔑`), jamais côté Reel (`👤`).
