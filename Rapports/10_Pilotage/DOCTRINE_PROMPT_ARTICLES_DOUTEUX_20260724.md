# PROMPT À TRANSMETTRE À DOCTRINE.FR (agent tiers « blanc »)

> Copier-coller intégralement le bloc ci-dessous dans Doctrine.fr.
> L'agent ne connaît RIEN du dossier : tout le contexte d'usage est fourni.
> Objectif : lever tout doute sur 18 références d'articles citées dans un dossier
> juridique, en donnant pour CHAQUE article le code exact, le numéro exact,
> le lien Légifrance (LEGIARTI), et la référence d'un arrêt de la Cour de cassation
> utile si pertinent.

## CONTEXTE D'USAGE (à lire en premier)

Vous êtes sur Doctrine.fr, une base de données juridiques française donnant accès
aux codes en vigueur (via Légifrance) et à la jurisprudence de la Cour de cassation
(et des cours d'appel). Votre rôle ici est de FOURNIR DES RÉFÉRENCES EXACTES ET
VÉRIFIABLES, pas de conseil juridique.

Pour chaque question ci-dessous, je veux de vous :
1. Le **code** exact (ex. : Code civil, Code pénal, Code de la sécurité sociale,
   Code général des impôts, Code de commerce, Code de la construction et de
   l'habitation, Code des relations entre le public et l'administration, etc.).
2. Le **numéro d'article exact** (avec son préfixe L./R./D. si applicable).

3. Le **lien Légifrance** vers l'article (format
   `https://www.legifrance.gouv.fr/codes/article_lc/LEGIARTIxxxxxxxx` — le numéro
   LEGIARTI est trouvable dans la barre d'adresse de Légifrance quand on ouvre
   l'article).
4. Le **texte de loi pertinent** (1 à 2 phrases).

5. Si l'article concerne une responsabilité / une infraction / une prescription,
   citez **un arrêt de la Cour de cassation** récent (moins de 10 ans) illustrant
   l'application de cet article, avec sa référence (ex. : Civ. 2e, 15 mai 1992,
   n° 90-14.261).

CONTRAINTE IMPORTANTE : dans mon dossier source, certains numéros sont ÉCRITS DE
FAÇON ERRONÉE (mauvais code rattaché, ou numéro inexistant tel quel). Votre job est
de trouver la BONNE référence. Si un numéro tel quel n'existe pas, dites-le et
proposez le bon article correspondant au sens recherché.

## LISTE DES 18 RÉFÉRENCES À VÉRIFIER

**1. Article « 29 »** — dans mon dossier il est écrit « article 29 de la loi
n° 85-677 du 5 juillet 1985 » (loi Badinter). Quel est le VRAI article de la loi
Badinter (loi n° 85-677 du 5 juillet 1985) qui traite de l'action directe de la
victime contre l'assureur ? Donnez le code + article + Légifrance + un arrêt.

**2. Article « 31 »** — écrit « article 31 de la loi n° 85-677 du 5 juillet 1985 »
(loi Badinter). Même question : quel est le bon article sur l'indemnisation /
l'action de la victime ? Code + article + Légifrance + arrêt.

**3. Article « 827 »** — écrit « article 827 du Code de procédure civile » dans le
contexte « partie demanderesse non assistée d'avocat ». L'article 827 du CPC
existe-t-il ? Si non, quel article du CPC traite de l'irrécusabilité / de
l'assignation sans avocat (ou de la représentation obligatoire) ? Donnez la bonne
référence + Légifrance + arrêt utile.

**4. Article « L. 725-3 »** — écrit « articles L. 725-3 à L. 725-4 du code rural et
de la pêche maritime » dans un courrier CPAM sur le recours subrogatoire de la
caisse. Le Code rural contient-il un L. 725-3 ? Si non, quel est le bon article du
Code de la sécurité sociale sur le recours de la caisse contre le tiers responsable
(ancien / nouveau numéro, car le code a été recodifié en 2019 dans le CSS) ? Donnez
code + article + Légifrance + arrêt.

**5. Article « L. 252 »** — écrit « Article L. 252 du Livre des procédures fiscales »
dans un courrier URSSAF. Existe-t-il un « Livre des procédures fiscales » ? Le bon
code est-il le Code général des impôts (CGI) ? Quel article du CGI traite de la
communication de renseignements entre administrations (ou de la mise en recouvrement
/ de la solidarité fiscale) ? Donnez code + article + Légifrance.

**6. Article « L. 274 »** — écrit « Article L. 274 du Livre des procédures fiscales »
dans le même courrier URSSAF (à propos de l'impôt et de la communication). Même
question : quel article du CGI (ou autre code fiscal) correspond ? Donnez code +
article + Légifrance.

**7. Article « L. 654-6 »** — écrit « Article L. 654-6 du Code de commerce »
(procédures collectives / dirigeants). L'article L. 654-6 du Code de commerce
existe-t-il en vigueur ? Si non (abrogé/recodifié), quel est le nouvel article sur
les sanctions des dirigeants de personne morale en procédure collective (banqueroute
ou fait de gestion) ? Donnez code + article + Légifrance + arrêt.

**8. Article « L. 224-98 »** — écrit « Article L. 224-98 » (supprimé selon mon
dossier). Confirmez qu'il n'existe plus et donnez, s'il y a lieu, l'article de
remplacement dans le Code de commerce (ou autre) sur le même sujet. Donnez
Légifrance de l'article abrogé + du remplaçant le cas échéant.

**9. Article « L. 210-6 »** — écrit « C. com. art. L.210-6 » dans une note sur
l'assurance des professionnels. L'article L. 210-6 du Code des assurances existe-t-il
? Si non, quel article du Code des assurances traite de l'application de l'assurance
légalement obligatoire (ex. : L. 211-1 sur l'obligation d'assurance de responsabilité
civile) ? Donnez code + article + Légifrance.

**10. Article « 1719-1720 »** — écrit « C. civ. art. 1719-1720 » (obligations du
bailleur). L'article 1719 du Code civil existe (obligation de délivrance du bailleur)
mais « 1719-1720 » comme plage existe-t-il ? Donnez les bons articles 1719 et 1720 du
Code civil (délivrance, entretien), Légifrance, et un arrêt sur l'obligation de
sécurité du bailleur envers le tiers (victime).

**11. Article « 25 »** — écrit « article 25 » (sans code) dans une note sur le droit
des sociétés et le capital social. De quel code s'agit-il (Code de commerce ?) et quel
est l'article exact sur le capital social / les apports ? Donnez code + article +
Légifrance.

**12. Article « R. 123-46 »** — écrit « article R. 123-46 » (sans code) dans une
demande de radiation d'une société au tribunal de commerce. De quel code s'agit-il
(Code de commerce ? Code de la construction et de l'habitation ?) et quel est
l'article exact sur la radiation d'office / la décision de justice rendue en matière
de difficultés des entreprises ? Donnez code + article + Légifrance.

**13. Article « 1358 », « 1362 », « 1369 », « 1378 », « 1353 »** — (si présents)
articles cités sans code dans des notes de prescription / preuve. Précisez le code
(Code civil sur la preuve / la prescription ?) et les bons numéros en vigueur
(rappel : le Code civil a été réformé en 2016, les anciens 1341-1369 sont devenus
des articles 1300-ish et 2224-2232). Donnez les bons numéros + Légifrance + arrêts.

**14. Article « L. 511-1 »** — écrit sans code dans une note sur la consommation /
assurances. De quel code (Code de la consommation ? Code des assurances ?) et quel
article exact ? Donnez code + article + Légifrance.

**15. Article « L. 654-1 »** — (si présent) du Code de commerce sur les dirigeants
de personne morale. Donnez code + article + Légifrance + arrêt.

## FORMAT DE RÉPONSE ATTENDU (une fiche par article)

Pour chaque point, répondez ainsi :

```
[Numéro litigieux] — libellé erroné trouvé : « ... »
  ✅ Code exact : <Nom du code>
  ✅ Article exact : <numéro>
  ✅ Lien Légifrance : https://www.legifrance.gouv.fr/codes/article_lc/LEGIARTIxxxx
  ✅ Texte (extrait) : « ... »
  ✅ Jurisprudence (si utile) : <juridiction, date, n°>, point de droit retenu
  ⚠️ Note : si le numéro d'origine était erroné, expliquez la correction.
```

Merci de traiter TOUS les points. Soyez exhaustif et citez vos sources (Légifrance
+ Doctrine.fr). Ne devinez jamais : si une référence est introuvable, dites-le et
proposez la bonne.
