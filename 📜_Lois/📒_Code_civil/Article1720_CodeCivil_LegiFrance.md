# Code civil — Art. 1720


Code civil
EN VIGUEUR DEPUIS LE 21/03/1804
Article 1720
VERSION EN VIGUEUR DEPUIS LE 21/03/1804
Création Loi 1804-03-07 promulguée le 17 mars 1804
Le bailleur est tenu de délivrer la chose en bon état de réparations de toute espèce.
Il doit y faire, pendant la durée du bail, toutes les réparations qui peuvent devenir nécessaires,
autres que les locatives.
