## 🔄 Workflow de mise à jour quotidien

1. **Lire ce README** — vérifier la cohérence des statuts
2. **Consulter STATUS.md** — comprendre où on en est
3. **Consulter VACCIN.md** — avant toute action
4. **Travailler** — modifications dans le repo local
5. **Générer versions réelles** : `python3 .dev/app/generate_real_versions.py`
6. **Git commit + push** : `git add -A && git commit -m "msg" && git push`
7. **Sync Drive NotebookLM** : `python3 .dev/app/sync_notebooklm.py`
8. **Mettre à jour ce README** si l'arborescence ou les statuts changent
