# Article 223-1

> **Nature** : Code

---

⚠️ **À VÉRIFIER** — Texte légal non récupéré automatiquement (accès Légifrance bloqué par Cloudflare dans l'environnement d'exécution ; pas de MCP Légifrance disponible ici).

À compléter via le MCP Légifrance (ou Judilibre) avec le texte officiel de l'Article 223-1 du Code pénal (Mise en danger d'autrui), conformément au protocole JURITEXT (Règle #16).

Référence attendue : Code pénal, art. 223-1 — « Le fait d'exposer directement autrui à un risque immédiat de mort ou de blessures de nature à entraîner une mutilation ou une infirmité permanente par la violation manifestement délibérée d'une obligation particulière de prudence ou de sécurité imposée par la loi ou le règlement est puni de un an d'emprisonnement et de 15 000 euros d'amende. »
