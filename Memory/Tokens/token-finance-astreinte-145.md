# 💰 Finance<br>Astreinte 145

**Token :** `**[Finance Astreinte 145]**`

## Valeur

| Champ | Valeur |
|---|---|
| **Montant** | 150 € / jour |
| **Fondement** | Article 145 du Code de procédure civile |
| **Variable canonique** | `MONTANT_ASTREINTE_145` |
| **Source** | [STRICT VARIABLES.md](../STRICT%20VARIABLES.md%23correspondance-tokens-financiers-alternatifs-finance) |

## Contexte

Astreinte journalière sollicitée dans le cadre de la requête sur le fondement de l'article 145 CPC, pour contraindre la production des pièces. Une alternative à 100 €/jour existe (MONTANT_ASTREINTE_REFERE).
