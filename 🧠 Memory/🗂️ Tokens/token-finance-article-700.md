# 💰 Finance<br>Article 700

**Token :** `**[Finance Article 700]**`

## Valeur

| Champ | Valeur |
|---|---|
| **Montant** | 3 000 € |
| **Fondement** | Article 700 du Code de procédure civile |
| **Tier** | Compromis |
| **Variable canonique** | `MONTANT_ARTICLE_700` |
| **Source** | [STRICT VARIABLES.md](../STRICT%20VARIABLES.md#correspondance-tokens-financiers-alternatifs-finance) |

## Contexte

Frais irrépétibles demandés au titre de l'article 700 CPC. Correspond au compromis (3 000 €). Une version conservative existe à 2 250 € (ART_700_CONSERVATIF).
